Sets the delta tuning value for the primitive at the specified index. This function will change the movement duration by the delta value on every update call thereby effectively slowing the movement down. The delta tuning value aims to achieve a damping motion, however this is not entirely functional. Using the delta argument will require a lot of fiddling.
